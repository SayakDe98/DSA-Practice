You are given a list of n strings a1, a2, a3,..., an.
You would like to concatenate them together in some order
such that the resulting string would be lexicographically smallest.
Given the list of strings, output the lexicographically smallest concatenation.

Input:
3
c 
cb
cba

Output: cbacbc

Input:
3
a
ab
aba

Output: aabaab

a, ab, aba
a, ab | aba
a, ab | aba
"a", "ab" -> a, aba, ab
aabaab
